# Security Policy

## Supported Versions

We support security updates for the following versions:

| Version | Supported |
|---------|-----------|
| 2.0.x   | Yes       |
| < 2.0.0 | No        |

## Reporting a Vulnerability

We take the security of Kelan Security very seriously. If you find a vulnerability, please do NOT open a public issue. Instead, report it privately to our security team.

Please email your report to **security@kelan.io** with the following information:
- A description of the vulnerability
- Detailed steps to reproduce
- Potential impact or proof of concept (PoC)

We will respond within 48 hours to acknowledge your report. If the vulnerability is verified, we will coordinate a fix and publish a security advisory.

Thank you for helping keep Kelan Security safe!
